# KC Wolff-Ingham Portfolio

Design Leadership Portfolio built with Astro 5.0. This project showcases 30 years of design leadership through an interactive resume and detailed case studies.

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
